This repository contains a React + TypeScript portfolio scaffold (Vite, Tailwind CSS).

Quick start
1. Install dependencies:
   npm install
2. Run the dev server:
   npm run dev
3. Open http://localhost:3000

Notes
- Tailwind is already referenced in src/styles/global.css. The build requires PostCSS + Tailwind (configs included).
- Add public assets used by the app to the project root or a public/ folder: `emmanuel-kimutai.jpg`, `Resume_Emmanuel Kimutai.pdf`.
- Replace the Formspree action in `src/pages/Contact.tsx` with your real form endpoint.

If you'd like, I can open a PR instead of committing directly, or add GitHub Actions to build/deploy.
